iFAST API Release 49

XML API

The iFAST XML API provides an XML interface to the iFAST Business Process Server. This API provides maintainace and inquiry access the the iFAST record keeping system.

  • The "iFast_Package" document type definition is located at docs/userdoc/iFast_Package.dtd

    • Direct Access Inquiry API

    The iFAST Direct Access API is a java based API for directly accessing iFAST data. This API supports inquiry access only (maintainace tasks are provide by the XML API outlined above).
