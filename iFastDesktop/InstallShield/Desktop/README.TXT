

From Release R55.0.0, Reflection will no longer be included with the iFast/Desktop releases.As a result of this change, the screen for Reflection selection has been removed from the Desktop installation process. So that the client will not be able to install Reflection at the same time of installing Desktop. Reflections for clients who uses it will be a separete download. Following are the process of the Reflection installation:

Step1: Down load Reflection from iFast.ca

Step2: Unzip it

Setp3: Open folder Disk1

Step4: run SETUP.EXE

Step5: Follow the instruction in <Instruction of Reflection installation.doc> which is in the same folder of unzip Reflection files located to finish the Reflection installation
